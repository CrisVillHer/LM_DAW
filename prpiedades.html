propiedades css colores

para poder establecer colores en css, podemos hacelo de la siguiente manera:
  • Color: podemos establecer el color de texto de nuestra web
  • background-color: nos permite establecer el color de fondo de nuestra web o elementos

para poder pojer un color en un elemento CSS, podemos hacerlo de la sguientes maneras:
  • Palabra clave: Ejem: red, blue, green, yellow, orange
  • RGB (red, green, blue): ejem: rgb(46,58,205)
  • RGBA con capa alpha: la capa alpha establece la tonalidad mas clara o mas oscura. Tiene valores de entre
    0 (mas claro) o 1 (como mas oscura). ejem: rgba(25,40,90,0.5)
  • Hexadecimal: Valores que van del 0-9 y A-F. Ejem: #FF7930
  • Hexadecimal abreviado: Tomamos como valor prioritario el primer digito. Ejem: #FF7930 --> #F73

Unidades en CSS

Dentro de las unidades de medida que podemos utilizar en CSS, podemos destacar dos grupos:
  • Unidades Absolutas: las unidades absolutas son un tipo de medida fija, es decir, no cambian nunca, son ideales para medios
    que no son variables, puesto que son pooco flexibles a la hora de usarlos. Ejem: pulgadas, cm, picas, mm, pt.
  • Unidades relativas: son un tipo de medida mas comun en CSS puesto que dependen de otra resolucion y tienen una curva
    de aprendizaje aunque mas compleja, son ideales para diferentes tamaños y para hacer web responsive.
    Ejem: em(1em= 16px), pixel, %, rem, etc.
    • width:Se utiliza para determinar un tamaño de anchura de los elementos
    • heightdeterminar el tamaño de altura de los elementos

Modelos de cajas en CSS

Todos los elementos en HTML, cuando creamos nuestras páginas web, están dentro de lo que se conocen como cajas (div)
las cajas que vamos a utilizar seran principalmente como se ha comentado div, y tambien podemos usar elementos o
etiquetas semanticas ya predefinidas: nav, aside, footer, header, article, etc.
A partir de ahora en CSS cuando vayamos a maquetar, dejaremos de utlizar tablas por empezar a usar este tipo de cajas.
El motivo principal es que las tablas no son elementos muy flexibles para maquetar en html, y por lo tanto, nos limitan
el que podemos adaptar nuestras web a distintos tipos de dispositivos.
Las propiedades que podemos aplicar a las cajas y que tenemos que tener en cuenta son las siguientes:
  • Margin:Corresponde al margen existente entre el borde exterior de nuestra caja / elemento contenedor y nuestro navegador
  • Padding: Corresponde al margen interior de nuestra caja, es decir, al espacio existente entre el contenido y el borde
    de nuestro elemento contenedor
  • Contenido:Son las etiquetas que van a ir dentro de nuestra caja o elemento contenedor
