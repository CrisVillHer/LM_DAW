Sistemas informáticos / Fundamentos de Hardware


CPU (Unidad Central de Procesamiento):

Definición: La CPU es el componente principal de un sistema informático responsable de ejecutar instrucciones y realizar operaciones aritméticas y lógicas. Es conocida como el "cerebro" de la computadora.

RAM (Memoria de Acceso Aleatorio):

Definición: La RAM es un tipo de memoria volátil que almacena temporalmente datos y programas en ejecución. Permite un acceso rápido a la información, pero se borra cuando la computadora se apaga.

Disco Duro:

Definición: El disco duro es un dispositivo de almacenamiento no volátil que guarda datos de forma permanente en un sistema informático. Almacena el sistema operativo, aplicaciones y archivos del usuario.

Placa Base:

Definición: La placa base es la principal tarjeta de circuitos de un sistema informático. Conecta y proporciona comunicación entre diversos componentes como la CPU, la RAM, el disco duro y otros periféricos.

Sistema Operativo:

Definición: El sistema operativo es un software fundamental que gestiona los recursos del hardware y permite la interacción del usuario con la computadora. Coordina la ejecución de programas y proporciona una interfaz para utilizar el hardware.
